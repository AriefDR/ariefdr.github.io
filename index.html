Telco Customer Churn

Issued by Digital Talent x Binar Academy
    The growth of the telecommunications industry is currently very rapid. Customers have the right to choose the appropriate provider and can switch from the previous provider which is defined as Customer Churn. Customer churn prediction is important for companies to know to be able to map a business strategy to retain customers.
  • Scope of Work :
    • Create a machine learning model using a classification algorithm (supervised learning) to predict customer churn.
    • Exploratory Data Analysis to find out which variables influence customer churn or not and create data visualizations.
    • Using several models such as Logistic Regression, KNN, Naive Bayes and Decision Tree. Based on these four algorithms, the result is that the decision tree algorithm obtains the highest accuracy around 88%.

Titanic - Machine Learning from Disaster

Issued by Digital Talent x Binar Academy
    The sinking of the Titanic is one of the most infamous shipwrecks in history. On April 15, 1912, during her maiden voyage, the widely considered “unsinkable” RMS Titanic sank after colliding with an iceberg. Unfortunately, there weren’t enough lifeboats for everyone onboard, resulting in the death of 1502 out of 2224 passengers and crew. While there was some element of luck involved in surviving, it seems some groups of people were more likely to survive than others. In this challenge, we ask you to build a predictive model that answers the question: “what sorts of people were more likely to survive?”
  • Scope of Work :
    • Create a machine learning model using a classification algorithm (supervised learning) to predict which type of person is more likely to survive.
    • Exploratory Data Analysis to find out the distribution of data that affects whether a person is survive or not.
    • Using a decision tree algorithm produces the highest accuracy reaching 94%.
